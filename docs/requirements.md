# Project Requirements 📝

Tài liệu này xác định các yêu cầu kỹ thuật và chức năng cho phiên bản Portfolio hiện tại và tương lai.

## 1. Yêu cầu Chức năng (Functional Requirements)
- **Hero Section**: Hiển thị tên, vị định hướng nghề nghiệp (Title) và lời chào ấn tượng.
- **Projects Showcase**: Hiển thị danh sách dự án với hình ảnh, mô tả ngắn, Tech Stack và link (GitHub/Demo).
- **Skill Visualization**: Trình bày các kỹ năng kỹ thuật một cách trực quan.
- **About Me**: Giới thiệu bản thân, kinh nghiệm và tính cách làm việc.
- **Contact Form**: Cho phép HR/Client gửi tin nhắn trực tiếp hoặc hiển thị thông tin liên hệ.
- **CV Download**: Nút tải xuống CV phiên bản mới nhất (PDF).
- **Responsive Design**: Hoạt động hoàn hảo trên Mobile, Tablet và Desktop.

## 2. Yêu cầu Phi chức năng (Non-functional Requirements)
- **Performance**: Điểm Lighthouse > 90 cho mọi chỉ số (Performance, Accessibility, Best Practices, SEO).
- **Modern UI**: Thiết kế theo phong cách hiện đại (Minimalism, Bento Grid hoặc Glassmorphism).
- **Smooth Interaction**: Sử dụng animation (Framer Motion) để tăng trải nghiệm người dùng nhưng không gây rối mắt.
- **SEO Optimized**: Tối ưu Meta thẻ, tiêu đề, và hình ảnh để dễ dàng tìm thấy trên Google.
- **Fast Loading**: Tối ưu kích thước hình ảnh và code splitting.

## 3. Tech Stack Hiện Tại
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React, MUI Icons
- **UI Components**: Radix UI, Material UI (MUI)
